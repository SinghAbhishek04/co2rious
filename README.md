## What it does

CO‚ÇÇrious transforms abstract carbon numbers into relatable equivalencies. Instead of just "340g CO‚ÇÇ", it shows "That's like driving 2.1 km" or "49 phone charges".

## Features

- **40 activities** across Transportation, Food, Home & Digital categories
- **Country selector** with region-specific grid carbon intensity (10 countries)
- **Metric/Imperial** unit toggle
- **Comparison mode** to compare activities side-by-side
- **Time projections** showing daily habits as yearly totals
- **Share** results to Twitter, LinkedIn, or clipboard

## Tech Stack

- React 18 + Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React icons

## Quick Start

```bash
git clone https://github.com/SinghAbhishek04/co2rious.git
cd co2rious
npm install
npm run dev
```
